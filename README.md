# Análisis de Operaciones y Productividad 📊

Este repositorio contiene las herramientas, scripts básicos de automatización y plantillas de documentación orientadas a la optimización de líneas de producción y análisis de tiempos en entornos industriales.

## 📁 Estructura del Proyecto

El proyecto se encuentra actualmente en fase de desarrollo inicial:

- `docs/`: Manuales de usuario y guías de instalación de software complementario.
- `src/`: Scripts en desarrollo para el cálculo de eficiencia de equipos (OEE) y balanceo de líneas.
